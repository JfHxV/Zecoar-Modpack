**怎么游玩泽扣整合包**

1. 打开您的我的世界启动器（例如：PCL2、HMCL、BakaCL等附有导入整合包功能的启动器）
2. 打开导入整合包界面（我们不提供找到导入整合包界面的教程，请上网寻求帮助）
3. 拖入/导入您下载的整合包压缩文件
4. 等待安装完成
5. 待安装完毕后即可游玩
